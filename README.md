# Python - Tic-Tac-Toe Game

### Description:

Single player game against CPU opponent. Three-by-three grid with reference grid to help pick spots. Win by placing three of the chosen marks in a horizonal, vertical, or diagonal row.    <br />
- Python Game - Tic-Tac-Toe - CPU Opponent - Difficult 
 
## Languages and Programs used
- Python
- Visual Studio Code
